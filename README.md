# useDOMID

React hook for using a unique HTML DOM ID.

## Install

### NPM

```bash
npm install --save use-dom-id
```

### Yarn

```bash
yarn add use-dom-id
```

## Usage

```javascript
const useDOMID from 'use-dom-id'

const id = useDOMID('select')
```
